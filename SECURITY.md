# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| latest  | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability in claws, please report it by [opening a GitHub Issue](https://github.com/clawscli/claws/issues/new).

Please include:
- A description of the vulnerability
- Steps to reproduce the issue
- Any relevant logs or screenshots

We will respond as quickly as possible and work with you to address the issue.
